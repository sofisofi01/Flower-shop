# Проект: Flower Shop

### О проекте

Простенький сайт-визитка для магазина цветов, есть возможность добавить фотографии реальных букетов, цены, форму для связи

**Технологии**

При создании данного проекта были использованы технологии HTML5, CSS3, JavaScript.
